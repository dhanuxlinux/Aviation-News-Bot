# Aviation News Bot

Python bot that:
- fetches aviation news from RSS feeds,
- filters already processed articles using Supabase,
- uses Gemini to pick the most viral stories,
- sends the final picks to Telegram.

## RSS feeds used
- Aviation Week
- AeroTime
- Simple Flying

## Project structure

```bash
aviation-news-bot/
├── .github/
│   └── workflows/
│       └── run-bot.yml
├── app.py
├── requirements.txt
├── .env.example
├── .gitignore
└── README.md
```

## 1. Install

```bash
pip install -r requirements.txt
```

## 2. Environment variables

Create a `.env` file from `.env.example` and add your real values:

```env
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key
GEMINI_API_KEY=your_gemini_api_key
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
TELEGRAM_CHAT_ID=your_telegram_chat_id
GEMINI_MODEL=gemini-3.1-flash-lite-preview
```

## 3. Supabase table

Run this SQL in Supabase:

```sql
create table processed_articles (
  id bigint generated by default as identity primary key,
  url text unique not null
);
```

## 4. Run locally

```bash
python app.py
```

## 5. GitHub Actions setup

Add these repository secrets in GitHub:
- `SUPABASE_URL`
- `SUPABASE_KEY`
- `GEMINI_API_KEY`
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`
- `GEMINI_MODEL` (optional)

Then push the repo. The included workflow can run on schedule and manually.
